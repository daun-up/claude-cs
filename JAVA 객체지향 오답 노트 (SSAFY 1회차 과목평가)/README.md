### Java 객체지향 완전 정복 📚

SSAFY 1회차 Java 과목평가 오답 분석을 기반으로 만든 인터랙티브 학습 가이드입니다.

#### 사용 방법

`oop_study_guide.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

#### 구성

| 탭 | 내용 |
|---|---|
| 📊 오답 현황 | 취약 개념별 교육생 정답률 정리표 |
| 다형성·참조타입 | 동적 바인딩, Pass by Value 코드 예측 퀴즈 |
| 키워드 정리표 | static / final / 접근제한자 / 인터페이스 / 제네릭 |
| 상속·생성자 순서 | 부모→자식 생성자 호출 순서 다이어그램 + 실습 |
| 예외·Stream·기타 | finally 흐름, Stream API, 배열 clone, 문자열 연산 |
| 포스트잇 오개념 | 16개 핵심 오개념 vs 올바른 개념 카드 |

#### 주요 기능

- **클릭해서 정답 확인** — 코드 출력 결과를 직접 예측한 뒤 버튼으로 확인
- **오답 포인트 강조** — 시험에서 틀린 개념 카드에 별도 표시
- **포스트잇 뷰** — 오개념(취소선) vs 올바른 개념을 한눈에 비교

#### 커버 개념

`Polymorphism` `Pass by Value` `static / final` `접근제한자` `Interface` `Generic` `Lambda` `Exception finally` `Stream API` `2D Array clone` `상속 생성자 순서` `문자열 연산`
