AI·ML·DL 기초 + 회귀 & 평가 지표

AI/ML/DL 정의 · 데이터 구성 · R² · 혼동행렬 · 회귀

▼
개념 정의
AI · ML · DL의 정의
세 개념의 포함 관계와 차이점. AI가 가장 넓고, ML이 그 안에, DL이 ML 안에 속하는 구조.
AI ⊃ ML ⊃ DL 지도학습 비지도학습 강화학습
AI: 인간 지능 모방 ML: 데이터로부터 규칙 학습 DL: 신경망 기반 ML
데이터
데이터 구성 요소
ML 모델 학습에 필요한 데이터의 구성. 특성(Feature), 레이블(Label), 훈련/검증/테스트 분할.
Feature Label Train/Val/Test 전처리
일반적 분할: Train 70~80% Val 10~15% / Test 10~15%
회귀 평가
회귀 설명력 R² (결정계수)
회귀 모델이 데이터의 분산을 얼마나 설명하는지 나타내는 지표. 1에 가까울수록 좋음.
0~1 범위 분산 설명력 회귀 전용
R² = 1 - SS_res / SS_tot SS_res = Σ(y - ŷ)² SS_tot = Σ(y - ȳ)²
분류 평가
혼동행렬 (Confusion Matrix)
분류 모델의 예측 결과를 TP/FP/FN/TN 4칸으로 정리한 표. Precision·Recall·F1의 출발점.
TP/FP/FN/TN Precision Recall F1 Score
Precision = TP/(TP+FP) Recall = TP/(TP+FN) F1 = 2×P×R/(P+R)
모델 유형
회귀 (Regression)
연속적인 수치를 예측하는 지도학습. 집값·온도·주가 예측 등. 출력층에 활성화 함수 없음.
연속값 예측 MSE/MAE 선형회귀
선형회귀: y = wx + b 손실: MSE = (1/n)Σ(y-ŷ)²
02

신경망 핵심 + 학습 알고리즘

RSS · 신경망 · 경사하강법 · 로지스틱 회귀 · 손실함수

▼
통계
RSS (Residual Sum of Squares)
실제값과 예측값의 잔차 제곱합. MSE의 기반이 되는 통계량. 작을수록 모델이 데이터에 잘 맞음.
잔차 제곱합 MSE 기반
RSS = Σ(yᵢ - ŷᵢ)² MSE = RSS / n
구조
신경망 (Neural Network)
퍼셉트론을 여러 층으로 쌓은 구조. 입력층→은닉층→출력층. 활성화 함수로 비선형성 추가.
퍼셉트론 은닉층 활성화함수 역전파
출력 = f(Wx + b) f = ReLU, Sigmoid, tanh...
최적화
경사하강법 (Gradient Descent)
손실을 줄이는 방향으로 가중치를 조금씩 수정하는 최적화 알고리즘. 기울기의 반대 방향으로 이동.
학습률 η 기울기 SGD/Adam 배치
w ← w - η · ∂L/∂w η: 학습률 (너무 크면 발산)
분류 모델
로지스틱 회귀 (Logistic Regression)
이진 분류 모델. Sigmoid로 출력을 0~1 확률로 변환. 이름은 '회귀'지만 분류 알고리즘.
Sigmoid 이진분류 확률 출력 BCE
P(y=1|x) = σ(wx+b) σ(z) = 1/(1+e⁻ᶻ)
학습
손실함수 (Loss Function)
모델 예측과 정답의 차이를 수치화. 역전파의 출발점. 문제 유형에 따라 다른 함수 선택.
MSE MAE Cross Entropy BCE
회귀 → MSE / MAE 이진분류 → BCE + Sigmoid 다중분류 → CE + Softmax
03

RNN · NLP · Transformer · BERT · LSTM

순차 데이터 처리 모델과 NLP 핵심 아키텍처

▼
시퀀스 모델
RNN 원리
순서가 있는 데이터를 처리하는 신경망. 이전 출력을 다음 입력에 반영. 기울기 소실 문제 존재.
순차 처리 은닉 상태 hₜ 기울기 소실
hₜ = tanh(Wₕhₜ₋₁ + Wₓxₜ + b) 단기 기억만 → LSTM으로 해결
NLP 평가
NLP 성능평가 지표
언어 모델의 성능을 측정하는 다양한 지표. 문제 유형에 따라 적합한 지표 선택이 중요.
PPL(낮을수록↓) BLEU(번역) ROUGE(요약)
PPL = exp(Cross Entropy) ↓좋음 BLEU = n-gram 일치율 ↑좋음 ROUGE= 요약 겹침 비율 ↑좋음
핵심 아키텍처
Transformer 원리
Self-Attention으로 문장 전체를 병렬 처리. RNN의 순차 처리 한계 극복. 현대 LLM의 기반 구조.
Self-Attention 병렬 처리 Q·K·V 위치 인코딩
Attention(Q,K,V) = softmax(QKᵀ/√dₖ)V W_all = W₃W₂W₁ → 비선형 필수
사전학습 모델
BERT 원리
양방향 Transformer 인코더. 마스크된 단어 예측(MLM)으로 사전학습. 문장 이해·분류·QA에 강점.
양방향 MLM Fine-tuning 인코더
사전학습: Masked LM + NSP 파인튜닝: 태스크별 헤드 추가
장기 기억
LSTM 원리
Cell state + Gate 3개로 장기 기억을 제어. RNN의 기울기 소실 문제 해결. 번역·음성 인식에 활용.
Cell state Forget Gate Input Gate Output Gate
Forget: fₜ = σ(Wf·[hₜ₋₁,xₜ]+bf) 삭제 Input : iₜ·C̃ₜ 저장 Output: hₜ = oₜ·tanh(Cₜ) 출력

텍스트 생성 방식 · 거대 언어 모델 평가 · RLHF

텍스트 생성
디코딩 알고리즘
LLM이 다음 단어를 선택하는 방식. Greedy·Beam Search·Sampling 등 다양한 전략 존재.
Greedy Beam Search Top-k/Top-p Temperature
Greedy : 매 스텝 최대 확률 선택 Beam(k=3): 상위 k개 경로 유지 Temperature: 높을수록 다양한 출력
LLM 평가
거대 언어 모델의 평가
LLM 출력 품질을 측정하는 방법들. 사람이 직접 채점하기 어려워 자동 평가 방식 발전.
LLM-as-a-judge 벤치마크 PPL MMLU
LLM-as-a-judge: 고성능 모델이 채점 사람 평가 대비 비용·시간 절감 단점: judge 모델 편향 반영 가능

RLHF 보상 모델 PPO DPO
RLHF 단계: ① SFT (지도 파인튜닝) ② 보상 모델 학습 ③ PPO로 강화학습

CNN 원리 · 연산 · Pooling · Stride

이미지 처리 신경망의 핵심 구조와 연산

▼
아키텍처
CNN 원리
이미지에 특화된 신경망. 가중치 공유 + 지역 연결로 파라미터 절감. 층이 깊어질수록 고수준 특징 학습.
가중치 공유 지역 연결 수용 영역 AlexNet
Conv→ReLU→Pool 반복 → Flatten → FC → Softmax
핵심 연산
CNN 연산 (합성곱)
필터가 입력 위를 슬라이딩하며 원소별 곱셈 후 합산. 출력 크기는 입력·커널·stride·padding에 의존.
커널 특징 맵 파라미터 수
출력크기 = (입력-커널)/stride + 1 파라미터 = 필터수×(입력채널×K×K)+필터수
다운샘플링
풀링 (Pooling)
특징 맵 크기를 줄이는 연산. 학습 파라미터 없음. 위치 불변성(translation invariance) 획득.
Max Pooling Avg Pooling 위치 불변성
Max Pool 2×2: [1 3] [5 6] → 6 (최댓값 선택)
연산 제어
스트라이드 합성곱 (Stride)
필터 이동 간격. stride=2이면 두 칸씩 이동 → 출력 크기 절반. 풀링 대신 stride로 다운샘플링 가능.
이동 간격 출력 크기↓ Padding
출력 = (입력-커널)/stride + 1 stride=1: 32→28 (커널5) stride=2: 32→14 (커널5)

멀티모달 모델과 AI 생태계

▼
멀티모달
Vision-Language Models
이미지와 텍스트를 함께 처리하는 모델. 이미지 캡셔닝·VQA·이미지 검색 등 다양한 응용.
CLIP 대조 학습 이미지 인코더 텍스트 인코더
CLIP: 이미지-텍스트 쌍 유사도 학습 임베딩 공간에서 일치 쌍 가깝게 불일치 쌍 멀게 → 대조 학습
CLIP 업그레이드
SigLIP
CLIP의 Softmax를 Sigmoid로 교체한 모델. 쌍별 독립 판단으로 더 효율적·강건한 학습 가능.
Sigmoid Loss 쌍별 독립 CLIP 대비 효율↑
CLIP : Softmax (배치 전체 비교) SigLIP: Sigmoid (쌍별 독립 판단) → 배치 크기 의존성 감소
생성 AI
이미지 생성 모델
텍스트나 노이즈로부터 이미지를 생성하는 모델. GAN·VAE·Diffusion 모델로 발전.
Diffusion GAN Stable Diffusion DALL-E
Diffusion: 노이즈 추가 → 제거 학습 GAN: Generator vs Discriminator VAE: 잠재 공간 인코딩·디코딩
플랫폼
허깅페이스 (HuggingFace)
사전학습 모델을 공유·사용하는 오픈소스 플랫폼. Transformers 라이브러리로 BERT·GPT 등 쉽게 사용.
Model Hub Transformers Pipeline API Fine-tuning
from transformers import pipeline clf = pipeline("text-classification") clf("이 영화 정말 재밌어요!")
AI 인프라
AI 파운데이션 모델
대규모 데이터로 사전학습된 범용 모델. 다양한 태스크에 파인튜닝 가능. GPT·BERT·LLaMA 등.
사전학습 파인튜닝 GPT LLaMA
사전학습(대규모) → Foundation Model → Fine-tuning(소규모, 태스크별) → RLHF (선호도 정렬)
